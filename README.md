### Project Title: EduAcad- An Online Educational Learning Platform

Prject Overview
EduAcad is a modern online education Learning platfrm inspired from Udemy and Coursera built using React+Vite.
It allows users to browse courses, enroll in them, track learning progress, complete modules, and manage their profile.

The system simulates a real-world e-learning platfrom such as Udemy and Coursera. It has following features:
- Course Catalog : For the demonstration purpose, 50 courses are stored in JSON file. User can search any courses they are interested to learn.
- User Authentication: Guest user can only check the course but they cannot enroll in it. To let user enroll in the course and check their progress, user authentication is created. User needs to sign up first and then log in to enroll in courses. For this, localStorage is used. 
- Course Enrolment: Once user is signed in, they can enroll in courses. Each course contain several modules. User can watch course video and they can also mark complete if they are done studying particular module.

- Progress Tracking: Based on modules they have completed in each course, they can see their course progress. 

- Dashboard Analytics: 
Course browsing and enrollment
Module-based video learning
Progress tracking system
User profile management
Search functionality
Dashboard analytics


# Technology Stack
- Frontend: React.js
- Routing: React Router DOM
- Styling: CSS modules + Bootstrap
- Icons: React Icons
- Data Storage: JSON file for storing courses, modules and inital signed in users. LocalStorage for mock backend.
